Learning tag <code>

Definition

The <code> tag is used to define a piece of computer code. It is typically used to display code snippets or programming examples in HTML documents.

Sometimes, it's hard to find all the special strings in the code, and changing those strings to HTML Entities is time-consuming. We could visit this site: codebeautify

Example:

  • <pre><code>&lt;html&gt;</code></pre> will results: <html>

  • <pre><code>&lt;h1&gt;Hello World!&lt;/h1&gt;</code></pre> will results: <h1>Hello World!</h1>
