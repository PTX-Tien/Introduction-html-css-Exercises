CSS Reset

1. Tại sao phải reset CSS:

  • Default CSS Values: trình duyệt sẽ tự thêm vào các thuộc tính mặc định để hiển thị các thẻ HTML, tuy nhiên các trình duyệt khác nhau có thể các thuộc tính được thêm vào sẽ khác nhau.
  • Các thẻ có CSS mặc định làm khó khăn trong việc kiểm soát kích thước và tính toán kích thước so với thiết kế

2. Cách reset CSS:

Vào website reset-css để copy đoạn code vào file .css

Lưu ý: đặt file reset.css ở vị trí trên cùng trước khi sử dụng các CSS khác để tránh file reset sẽ ghi đè, do tứ tự ưu tiên về vị trí đặt.
