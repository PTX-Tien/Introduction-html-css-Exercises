System Font

1. System Font CSS - Công cụ bí mật tăng tốc độ trang web

Một số loại font chữ chính:

  • Serif - Chữ có chân
  • Sans-serif - Chữ không chân
  • Monospace - Các ký tự cùng khoảng rộng

Để tham khảo một số nguồn System Font, có thể vào trang system-font-stack

Có rất nhiều font chữ đẹp, tuy nhiên không phải font chữ nào cũng có sẵn trên máy khách -> Cần đặt font chữ nhất quán phòng cả trường hợp máy khách không có font chữ này. Để tham khảo một số loại font được sử dụng phổ biến trên các hệ điều hành, có thể tham khảo trang cssfontstack

Nếu máy khách không hỗ trợ loại font trên thì -> Nhúng font chữ

2. Nhúng Font:

Nhúng font chữ - sử dụng trên trang web để bất cứ máy nào cũng hiển thị giống nhau, kể cả trường hợp máy khách không cài đặt font này.

Để nhúng font, có thể vào trang google fonts
