CSS Box Model - Mô hình hộp CSS

Box Model: Trình duyệt luôn coi mỗi phần tử là 1 box - hộp chữ nhật

  • Nội dung (width, height) -> hình ảnh, text, video...
  • padding -> khoảnh đệm nội dung và viền (không chứa nội dung)
  • border -> viền của phần tử (viền nét liền, đứt...)
  • margin -> Khoảng cách với các phần tử liền kề khác

NOTE: Kích thước cao * rộng 1 phần tử = content + padding + border (Không bao gồm margin)
