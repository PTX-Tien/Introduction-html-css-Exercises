2. Internal CSS

Internal (CSS nội bộ) là cách viết mã CSS trong phần <head> của trang HTML

Cấu trúc

<head>
    <style>
        tagname {
            property: value;
        }
    </style>
</head>
  1. Ví dụ:
    • Dòng thứ 1
    • Dòng thứ 2
    • Dòng thứ 3
  2. Nhận xét:
    • Có thể thay đổi màu sắc của cả 3 dòng trên
    • Có thể tái sử dụng cho nhiều dòng khác nhau
    • Dễ bảo trì và cập nhật
    • Dễ áp dụng cho nhiều phần tử cùng lúc

Ưu điểm của Internal CSS

  • Dễ dàng quản lý và bảo trì
  • Cho phép áp dụng kiểu dáng cho nhiều phần tử cùng lúc
  • Giúp tách biệt nội dung và kiểu dáng
  • Tiết kiệm thời gian khi cần thay đổi kiểu dáng

Nhược điểm của Internal CSS

  • Không thể tái sử dụng trên các trang khác
  • Không thể áp dụng cho nhiều trang cùng lúc
  • Không thể sử dụng lại trong các dự án khác
  • Không thể áp dụng cho các phần tử bên ngoài trang HTML
