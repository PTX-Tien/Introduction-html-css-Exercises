3. External CSS

The external CSS file is linked in the <head> section of the HTML document to apply styles across multiple pages.

Example of Linking CSS

<link rel="stylesheet" href="style.css">

This line links the external CSS file named "style.css" to the HTML document. It allows for consistent styling across all pages that include this link.

Benefits of External CSS

  • Reusability: Styles can be applied to multiple pages without duplication.
  • Maintainability: Changes to the CSS file automatically reflect on all linked pages.
  • Separation of Concerns: Keeps HTML content separate from styling, improving readability.
  • Faster Loading: Browsers can cache the CSS file, reducing load times for subsequent page visits.

Conclusion

Using external CSS is a best practice in web development, allowing for efficient and organized styling of web pages. It enhances the user experience by providing a consistent look and feel across the site.
