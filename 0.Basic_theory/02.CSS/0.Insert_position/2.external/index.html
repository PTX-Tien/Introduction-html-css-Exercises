Vị trí đặt CSS

3. External CSS

External CSS là một tập tin CSS riêng biệt, được liên kết với trang HTML thông qua thẻ <link />. Điều này cho phép bạn áp dụng cùng một kiểu dáng cho nhiều trang web khác nhau.

Cấu trúc

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="style.css" />
    <title>Document</title>
</head>
  1. Ví dụ:
    • Dòng thứ 1
    • Dòng thứ 2
    • Dòng thứ 3
  2. Nhận xét:
    • Có thể thay đổi màu sắc của cả 3 dòng trên
    • Có thể tái sử dụng cho nhiều dòng khác nhau
    • Dễ bảo trì và cập nhật
    • Dễ áp dụng cho nhiều phần tử cùng lúc

Ưu điểm của External CSS

  • Dễ dàng quản lý và bảo trì
  • Cho phép áp dụng kiểu dáng cho nhiều trang web cùng lúc
  • Giúp tách biệt nội dung và kiểu dáng
  • Tiết kiệm thời gian khi cần thay đổi kiểu dáng

Nhược điểm của External CSS

  • Yêu cầu tải thêm một tập tin CSS, có thể làm chậm tốc độ tải trang
  • Cần phải đảm bảo rằng tập tin CSS được liên kết đúng cách
  • Không thể áp dụng kiểu dáng cho các phần tử cụ thể trong trang HTML
  • Cần phải có kiến thức về cách sử dụng thẻ `` để liên kết tập tin CSS
