Vị trí đặt CSS

1. Inline CSS

Inline (CSS nội tuyến) là cách viết mã CSS ngay bên trong phần tử cơ bản HTML

Cấu trúc

<tagname style="property:value;">Content</tagname>
  1. Ví dụ:
    • Thay đổi ở dòng 1
    • Thay đổi ở dòng 2
    • Thay đổi ở dòng 3
    • Thay đổi ở dòng 4
  2. Nhận xét:
    Chỉ có thể thay đổi màu sắc trên từng dòng một, rất mất thời gian khi muốn tác động đồng thời cả 4 dòng trên.

Nhược điểm của Inline CSS

  • Không tách biệt được nội dung và kiểu dáng
  • Không thể tái sử dụng
  • Khó bảo trì và cập nhật
  • Không thể áp dụng cho nhiều phần tử cùng lúc
