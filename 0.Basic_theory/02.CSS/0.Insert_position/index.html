CSS

Definition

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout, colors, fonts, and overall appearance of web pages.

Key Concepts

  • Selectors: Patterns used to select elements to apply styles.
  • Properties: Attributes that define how elements should be styled.
  • Values: Specific settings for properties (e.g., color, font-size).

Insert Position

The insert position in CSS refers to where new styles or elements are added within the document structure. This can affect how elements are displayed and interact with each other.

There are 3 main ways to insert CSS into HTML (inline, internal, and external):

  1. Inline
  2. Internal
  3. External
