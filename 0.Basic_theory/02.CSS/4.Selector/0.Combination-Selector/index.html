Combination Selectors

1. * Selector

The * selector selects all elements in the document. It is a universal selector that applies styles to every element.

Example Usage

* {
    color: blue;        /* Applies blue color to all text */
    font-size: 30px;    /* Sets font size to 30px for all elements */
}

2. Element.class Selectors

The element.class selector selects elements of a specific type that also have a specific class. This allows you to style only those elements that match both criteria.

Example Usage

p.highlight {
    background-color: yellow; /* Highlights paragraphs with the 'highlight' class */
}

3. Element, Element

The element, element selector selects multiple elements of the same type. This allows you to apply the same styles to different elements without repeating the style rules.

Example Usage

h1, h2, h3 {
    color: green;                     /* Applies green color to all headings */
    font-family: Arial, sans-serif;   /* Sets font family for all headings */
}

Bộ chọn element element là bộ chọn kết hợp giữa hai phần tử, trong đó phần tử đầu tiên là phần tử cha và phần tử thứ hai là phần tử con. Bộ chọn này cho phép bạn áp dụng kiểu cho các phần tử con của một phần tử cha nhất định.

6. Element > Element

Bộ chọn element > element cho phép bạn lựa chọn các phần tử con trực tiếp của một phần tử cha. Bộ chọn này giúp bạn áp dụng kiểu cho các phần tử con mà không áp dụng cho các phần tử con của các phần tử con.

Giả sử div > p là bộ chọn kết hợp giữa phần tử div và phần tử p. Bộ chọn này sẽ áp dụng kiểu cho tất cả các phần tử p trực tiếp bên trong một phần tử div.

Giả sử h1 + p là bộ chọn kết hợp giữa phần tử h1 và phần tử p. Bộ chọn này sẽ áp dụng kiểu cho tất cả các phần tử p ngay sau một phần tử h1.
