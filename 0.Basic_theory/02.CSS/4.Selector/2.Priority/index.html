  1. !important
  2. Inline style (Viết trên code HTML)
  3. ID (#) selector
  4. Class (.) hoặc Pseudo-class (: ) selector
  5. Element selector (h, p, li, div,...)
  6. Universal selector (*) : Bộ chọn chung
