Learning tag <table>

Definition

The <table> tag is used to create a table in HTML. It allows you to organize data in rows and columns.

Basic Structure

<table>
    <thead> 
        <tr> 
            <th>Header 1</th> 
            <th>Header 2</th>
            <th>Header 3</th>
        </tr>
    </thead>
    <tbody>
        <tr> 
            <td>Data 1</td> 
            <td>Data 2</td>
            <td>Data 3</td>
        </tr>
        <tr> 
            <td>Data 4</td>
            <td>Data 5</td>
            <td>Data 6</td>
        </tr>
    </tbody>
    <tfoot> 
    <tr> 
        <td>Data 7</td>
        <td>Data 8</td>
        <td>Data 9</td>
    </tr>
    </tfoot>
</table>

Example Table

Header 1 Header 2 Header 3
Data 1 Data 2 Data 3
Data 4 Data 5 Data 6
Data 7 Data 8 Data 9

Learning Colspan and Rowspan

The colspan attribute allows a cell to span multiple columns, while the rowspan attribute allows a cell to span multiple rows. 

<table>
    <tr>
        <td colspan="2">This cell spans 2 columns</td>
        <td>Normal Cell</td>
    </tr>
    <tr>
        <td rowspan="2">This cell spans 2 rows</td>
        <td>Normal Cell</td>
        <td>Normal Cell</td>
    </tr>
    <tr>
        <td>Normal Cell</td>
        <td>Normal Cell</td>
    </tr>
</table>
